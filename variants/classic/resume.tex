\project{Cluster-Constrained Portfolio Optimization}{Python, Scikit-learn}{2025 -- 2026}
\begin{itemize}
  \item Researched whether sector-clustering constraints improve mean-variance portfolio performance vs.\ plain MV on a 33-stock sector-stratified universe.
  \item Built an end-to-end backtesting pipeline: Ward hierarchical clustering $\rightarrow$ cluster-constrained MV optimization $\rightarrow$ benchmarking against Plain MV and ERC across 860 rolling 5-year windows.
  \item Showed cluster-constrained MV matches plain MV's out-of-sample Sharpe while materially lowering portfolio concentration, max drawdown, and 5\% expected shortfall.
  \item Identified that plain MV behaves as an \emph{error maximizer}, concentrating allocations in in-sample winners; cluster constraints mitigate this without sacrificing risk-adjusted returns.
\end{itemize}
